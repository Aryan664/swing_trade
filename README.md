# Swing Trade App

A professional-grade swing trading educational application using real-time market data.

## Getting Started

### 1. Start the Backend
```bash
cd backend
npm install
node server.js
```

### 2. Start the Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- **$10,000 Paper Trading:** Start with mock funds and practice swing trading.
- **Real-Time Data:** Fetches live stock prices using the Yahoo Finance API.
- **Portfolio Management:** Track your holdings, average entry prices, and total investment.
- **Brokerage-Grade UI:** High-fidelity dark mode interface built with Tailwind CSS.

## Technology Stack
- **Frontend:** React, Vite, Tailwind CSS, Lucide Icons.
- **Backend:** Node.js, Express, Yahoo Finance API.
- **Database:** Local `db.json` for persistence.
